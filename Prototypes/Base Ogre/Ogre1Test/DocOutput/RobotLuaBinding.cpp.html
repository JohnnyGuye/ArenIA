getTerrainSize():int,int

Description

Returns the height and the width of the Terrain

Return

- height : The heigh in units of the terrain - width : the width in units of the terrain

getWallSize():int

Description

Returns the size of the map's walls. The walls are squares, so this function returns both width and length of the

walls at the same time.

Return

- the size of the walls.

turnRobot(angle:double):void

Description

Gives the robot the order to turn at a certain angle

Parameters

- angle:double The angle in degree that the robot has to turn. A positive angle means he will turn anti-clockwise, and a negative one means he will turn anti-clockwise.

Example

How to turn

    function main()
        turn(50)
        -- Now the robot will turn in order to have a angle of 50 degres
    end

useAbility(id:int):bool

Description

Use the ability with the given id

Parameters

- id An int corresponding to the number of the ability to use. Call getKnownAbilities in order to know what abilities are aviable

Return

- A bool that is true if the robot has an ability for that index, and false otherwise

getStats(id:int):table

Description

Get the currents and max stats of the robot with the given id in a lua table. If no robot is given, your robot is chosen. 

The lua table contains these fields
<ul>
    <li>maxHp : The max hp of the robot </li>
    <li>maxHp : The max hp of the robot</li>
    <li>maxEnergy : The max Energy of the robot</li>
    <li>currentEnergy : The current Energy of the robot </li>
    <li>range : The range of the weapon</li>
    <li>resistance : The armor of the robot</li>
    <li>speed : The speed of the robot</li>
</ul>

Example

How to use getStats()

    function main()
        -- Get the stats of your robot
        stats = getStats()

        -- Get the stats of antother robot
        statsOtherRobot = getStats(1)
        -- Use them
        if (stats.currentHP < 10) then
            -- ...
        end
    end

getRobots():table

Description

Returns a table containing the list of all the robots in the game.

Your robot will always be in the first (1) index.

isAWall(x:int,y:int):bool

Description

Tells if there is a wall at the given coordinate.

Parameters

- x The x position of the point where to check - y The y position of the point where to check

Return

- True if there is a wall, false if there is no wall
